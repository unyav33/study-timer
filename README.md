The study-timer project is a full-stack Web App, built for my full-stack learning process.
This web app was created from tutorial by google gemini. It includes these following fuctions for the first version:
1. Frontend
   - Countdown timer
   - Start, Pause, Stop button
3. Backend
   - POST: save sessions
   - GET: show history
4. Database
   - Date time
   - Session Table
